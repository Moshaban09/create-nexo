## How to Contribute

### 1. Fork the Repository
Fork [create-nexo](https://github.com/Moshaban09/create-nexo) to your GitHub account.

### 2. Clone Your Fork
```bash
git clone https://github.com/YOUR_USERNAME/create-nexo.git
cd create-nexo
npm install
```

### 3. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes
- Follow the existing code style
- Add tests for new features
- Update documentation if needed

### 5. Test Your Changes
```bash
npm run build
npm run test:run
```

### 6. Commit & Push
```bash
git add .
git commit -m "feat: your feature description"
git push origin feature/your-feature-name
```

### 7. Open a Pull Request
Go to the original repository and open a PR.

---

## Development Setup

```bash
npm install
npm run dev    # Watch mode
npm run build  # Build
npm run test   # Run tests
```

## Code Style

- TypeScript with strict mode
- ESM modules
- Meaningful variable names
- Comments for complex logic

## Reporting Issues

Please use [GitHub Issues](https://github.com/Moshaban09/create-nexo/issues) for:
- Bug reports
- Feature requests
- Questions
